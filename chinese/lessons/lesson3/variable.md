## 什么是变量？

字面来讲就是变化的量，额额额，讲左等于冇讲。。。

比如有个某天的气温，它会随着时空的变化而变化，「这天的气温」就是一个变量。

在程序中我们通常使用类型加表达式来创建一个变量，比如：

```dart
int temp = 30;
```

这里 `int` 是类型，`temp` 是变量名，`30` 是表达式。

这样创建一个变量的行为，我们称之为「定义」。（或者叫「声明」）

好像有些难懂。。。来看看下面的例子：

```dart
int temp = 30;
print(temp);
```

这时候会打印出 `30`，而不是 `temp`。
