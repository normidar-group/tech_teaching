# 計算


## 計算は3種類の計算がある


## 四則計算

- `+ - / *` の四つの計算が四則演算です

四則演算は数字から数字へ計算です

たとえば：

```dart
int a = 1 + 1;
double b = 1.5 + 2.1;
```

## 比較計算

- `> < >= <= !=` の五つがある

比較演算は数字からboolへ計算するものです。

たとえば：

```dart
bool a = 3 > 2;  // -> 結果はtrueになる
bool b = 3 < 1;  // -> 結果はfalseになる 
```
